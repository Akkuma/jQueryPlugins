jQuery Print Element 1.0 is released

After quite a while, jQuery.printElement has now been updated to version 1.0

Bug fixes:

  • IE 8 no longer alerts about script beeing closed
  • Form inputs are now printed as well
  • Due to some issues with Opera and Chrome 3.0, these browsers have been limited to popup mode only
  • The sample page was updated
